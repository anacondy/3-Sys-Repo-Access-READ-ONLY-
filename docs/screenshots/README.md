# Screenshots Directory

This directory is for storing screenshots of The Archives application.

## Expected Screenshots

- `landing.png` - The cinematic landing page
- `grid-view.png` - File explorer grid view with thumbnails
- `player.png` - Built-in media player with controls
- `mobile.png` - Mobile device responsive layout

## How to Add Screenshots

1. Take screenshots while using the application
2. Save them to this directory with appropriate names
3. Commit and push the changes

Screenshots should be reasonable resolution (around 1280x720 for desktop, appropriate sizes for mobile).
